## Lists

Ordered and unordered lists will be converted to HTML:

1. This is item one
2. This is item two

- This is item one
- This is item two

## Code

You can add in-line using single tics.

Create a varible named `var test = 1`.

Or you can use three tics for a block of code:

```javascript
var test = 1;
if(test == 1)
{
    console.log("Hello World!");
}
```
